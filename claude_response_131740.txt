```bash

```

または、メインのPythonファイルを探すために：

```bash
# メインファイルを探す
ls -la *.py | grep -E "(main|app|run|start)"
```

```bash
# agentsディレクトリの内容を確認
```

```bash
# READMEやドキュメントの内容を確認

```
